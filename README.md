# Network Topology Visualizer

An interactive network topology visualizer built using react and cytoscape.js

# Its features include:-

- adding router, switches and PCs
- It auto connects automatically to the selected nodes
-Can be deleted, saved and load too with JSON file

I built this initially to learn about cytoscape.js and other fundamentals that includes in the javascript, also i was bored and wanted to try something new in this summer holidays so i learn and built this basic functional website in just 2 days.
